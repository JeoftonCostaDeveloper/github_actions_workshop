# Github Actions Workshop
Repositório Modelo Treinamento Github Actions

## Roteiro
### O que é _CI/CD_

Explicação do que seria CI/CD.

#### Continuous Integration

Prática de desenvolvimento no qual há integração diária de código em um repositório compartilhado. Neste prática cada integração é verificada por um build e testes automatizados.

#### Continuous Delivery

Prática que estende o CI, automatizando a entrega do código em ambientes remotos dev, test, hml ou prd.
